Adds a custom Get()/Set() property to the type. See main.cpp for example usage.

Parameters
NameThe name of the property as you would like it to appear (no quotes!)
getlambdaMust either be a function pointer or a lambda that returns the desired property. If the pointer is to a member function of the reflected type, then it must take no parameters. If it's a lambda or a pointer to a non-member function, then it must take one parameter, either as the type itself by value, pointer, or reference, or as a variant_type& or variant.
setlambdaSimilar to getlambda, but the set function doesn't have to return anything. If it returns a bool, true should represent a success and false a failure. This return value will be passed along in property::set() and variant::assign() if applicable. Like getlambda, if this is a nonmember function it must take the this pointer either as the correct type or as a variant. The last parameter of the set function is how the value to assign is passed in, again either as the correct type or as a variant.
